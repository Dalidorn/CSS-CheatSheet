CSS Snippet Cheatsheet

Easy cheatsheet for all the CSS tricks I learn! Each card has copy and paste-able code for ease of use!

Flexbox Row

The following CSS will cause the .row class to be configured as a horizontal row!

                            .row {
                                display: flex;
                                flex-direction: row;
                                flex-wrap: wrap;
                            }

Flexbox Column

The following CSS will cause the .column class to be configured as a vertical column!

                            .column {
                                display: flex;
                                flex-direction: column;
                            }

CSS Grid

It's possible to build a grid layout in CSS, the following with make a 12 column layout under class grid.

                            .grid {
                                display: grid;
                                width: 100%;
                                grip-template-columns:
                                    repeat(12, 1fr);
                            }

Linear Gradients

The Linear Gradient function can be used in conjunction with background-image to create gradient backdrops.

                            .background {
                                background-image:
                                    linear-gradient(
                                    COLORHERE 0%,
                                    COLORHERE 100%
                                    )
                            }

Box Transition Glow

Mixing transition and box-shadow it's possible to make a glow effect. In this case, when hovering over the code card, it and the header glow.

                            .code-card .card-header {
                                boarder-radius: 8px;
                                transition: all 0.5s
                                ease-in-out;
                            }

                            code-card:hover,
                            code-card:hover .card-header {
                                box-shadow: inset
                                    0 0 8px COLORHERE,
                                    0 0 15px COLORHERE
                            }

Overlay Card with Title

Using margins it's possible to title cards!

                            .card-header {
                                position: relative;
                                margin-top: -20px;
                            }
